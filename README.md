# CS2 CustomIO For SwiftlyS2
Fixes missing keyvalues from CSS/CS:GO
From https://github.com/darkerz7/CS2-CustomIO

## Features:
1. Adds a custom Input-Output as in CSS/CS:GO: [From cs2fixes](https://github.com/Source2ZE/CS2Fixes/blob/main/src/customio.cpp)

## Required packages:
1. [SwiftlyS2](https://github.com/swiftly-solution/swiftlys2/releases) (Min version: v1.1.1-beta.54)

## Installation:
1. Compile and copy cs2-CustomIO to `swiftlys2/plugins/cs2-CustomIO` folder
2. Copy `resources` folder to `swiftlys2/plugins/cs2-CustomIO` folder
3. Restart server
